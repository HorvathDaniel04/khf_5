# Formai követelmények

Figyelj a beadás formai követelményeire!

> [!IMPORTANT]
> A megoldásokat pull request formájában kell beadni a határidő előtt. Kis házi feladatok esetén a határidőt lásd a tárgykövetelmények között (előadás diasor).
> - Hozz létre egy **új branchet** `megoldas` néven, és ezen dolgozz.
> - Töltsd ki a repository gyökér könyvtárában lévő `neptun.txt` fájlt a saját Neptun kódoddal.
> - A feladat végeztével vagy a labor végén **pushold** a megoldásodat és hozz létre egy **pull requestet**.
> - Ellenőrizd a pull request tartalmát és rendeld hozzá a laborvezetődet **reviewernek**.
> - A Git használatáról részletesebben az [első labor anyagában](https://bme-vik-informatika2.github.io/laborok/laborok/01-git/) vagy [itt](https://bme-vik-informatika2.github.io/laborok/tudnivalok/github/GitHub-classroom/) olvashatsz.

> [!CAUTION]
> A nem ilyen formában megadott megoldások nem lesznek értékelve!
